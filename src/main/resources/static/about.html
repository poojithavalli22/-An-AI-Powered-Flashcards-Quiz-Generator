About This Project

What is AI Synapse prep?

AI Synapse Prep transforms your text or PDF content into structured flashcards and interactive tests using your Spring Boot backend and Google Gemini. It’s optimized for learning with fast generation, a clean UI, and professional exports.

Key Features

  • Generate flashcards from text or PDF
  • Multiple question types: MCQ, True/False, Fill-in-the-Blank, Short, Long
  • Interactive timed tests with results and review
  • Export flashcards and results to PDF

Tech Stack

Spring Boot, RESTful API, WebClient, Google Gemini, HTML5, CSS3, JavaScript, jsPDF.
